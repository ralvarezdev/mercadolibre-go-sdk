## Gestionar IPs de una aplicación

 Importante: 

Esta funcionalidad es exclusiva a integradores withelisteados por Mercado Libre.

Esta documentación tiene la finalidad de mostrar la funcionalidad disponible desde el [devcenter](https://developers.mercadolibre.com.ve/devcenter/) para gestión y configuración de rangos de IP que serán permitidos para el consumo de nuestras APIs. Describiendo los posibles flujos que un usuario puede ejecutar desde esta nueva pantalla.

## Configuración IP

Para ingresar a las aplicaciones el usuario administrador debe estar logueado al [devcenter](https://developers.mercadolibre.com.ve/devcenter/). Por cada aplicación integrada encontrarás un menú desplegable, en donde se visualizará la opción de **Configuración IP**.

Nota:

- Si 

Configuración Ip

 no se lista es porque la aplicación no está habilitada para realizar gestión de IPs. 

## Listado de rangos

En la parte inferior de la pantalla, se muestra un listado de las IPs configuradas en la aplicación. En la barra de búsqueda se puede digitar el rango que desea encontrar de una manera más rápida:

## Agregar nuevas IPs

En la parte derecha de la pantalla se muestra la cantidad de IPs que se agregaron para la aplicación y la cantidad de IPs disponibles. Si aún tiene rangos disponibles se habilitará la sección para agregar una nueva IP.

Existe la posibilidad de agregar nuevas IPs de dos maneras:

1. Agregando la IP de forma individual.
2. De forma masiva, cargando un archivo con extensión .csv con el listado de IPs a agregar.
