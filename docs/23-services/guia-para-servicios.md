# Guía para Servicios

Los servicios son publicaciones de clasificados donde los datos de contacto de los prestadores son públicos, lo que facilita el contacto directo entre un usuario y el prestador del servicio, y el pago y las condiciones de contratación se coordinan entre las contrapartes. Puedes crear aplicaciones que publiquen servicios y tomen métricas de los contactos, realicen búsqueda de servicios por geolocalización, comparen precios y prestaciones, envíen recordatorios, sugieran similares, etc.
