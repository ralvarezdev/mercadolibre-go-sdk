## Paquetes y permisos para proyectos, desarrollos o emprendimientos inmobiliarios

En **Mercado Libre**, una publicación de **Desarrollo, Proyecto o Emprendimiento Inmobiliario** permite a los vendedores mostrar unidades en construcción o por construir. Cada unidad puede ofrecer distintas alternativas, llamadas **variaciones**, que especifican una posible unidad a construir, incluyendo sus características, planos y otros detalles.

Si deseas conocer más acerca de este tipo de publicaciones, consulta la guía de [Desarrollos inmobiliarios](https://developers.mercadolibre.com.ar/es_ar/test-guia-inmuebles). Para poder crear publicaciones de inmuebles con variaciones, se necesitan **permisos adicionales** y un **paquete especial**. Si necesitas estos privilegios, puedes solicitarlos a través del [formulario de contacto](https://forms.gle/nmnCFkikCJKJcaec9), seleccionando la opción de **privilegios y paquete de desarrollo**.

Una vez que te confirmen que se han otorgado los permisos y activado el paquete, podrás realizar las publicaciones. Sigue la [guía de variaciones](https://developers.mercadolibre.com.ar/es_ar/variaciones-para-inmuebles) para más detalles.

Importante

- Este tipo de paquete se activa por una sola cuota y permite una única publicación con variaciones. Para más publicaciones, se requerirá una nueva solicitud.
- La activación de este paquete es específica para un usuario, ya que solo podrá realizar publicaciones con variaciones, no publicaciones convencionales.
